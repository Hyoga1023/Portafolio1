Skills

En mi día a día, trabajo con HTML, CSS y JavaScript, los pilares fundamentales del desarrollo web. Con HTML, estructuro el contenido de las páginas, creando una base sólida que permite que la información sea accesible y semánticamente correcta. CSS es mi aliado para dar vida a estas estructuras, diseñando interfaces atractivas y funcionales que mejoran la experiencia del usuario. A través de JavaScript, añado interactividad, permitiendo que las páginas respondan a las acciones del usuario y se comporten de manera dinámica.

A medida que profundizo en mis habilidades, espero ampliar mi arsenal aprendiendo Node y React. Node me permitirá desarrollar aplicaciones del lado del servidor, dándome la capacidad de crear soluciones completas y eficientes que mejoren el rendimiento de mis proyectos. Por otro lado, React promete llevar mis interfaces a un nuevo nivel, facilitando la creación de componentes reutilizables y optimizando la gestión del estado de las aplicaciones. Con estas herramientas, mi objetivo es construir experiencias web más complejas y envolventes, fusionando diseño y funcionalidad de manera más efectiva.

Proyectos Destacados

En mi trayectoria como desarrollador web, he trabajado en proyectos que fusionan creatividad, interactividad y funcionalidad. A través de HTML, CSS y JavaScript, he logrado crear experiencias digitales dinámicas que no solo cumplen con los requerimientos técnicos, sino que también ofrecen una estética moderna y atractiva. A continuación, presento dos proyectos destacados que demuestran mi habilidad para implementar soluciones innovadoras, tanto en el ámbito de la seguridad digital como en la creación de contenido interactivo para entretenimiento.

Encriptador de Texto: Desarrollo de una aplicación web interactiva que permite cifrar y descifrar textos utilizando HTML, CSS y JavaScript. El proyecto cuenta con una interfaz moderna y dinámica que optimiza la experiencia del usuario. El encriptador permite introducir un mensaje en una columna y visualizar su versión cifrada en la columna opuesta. Además, implementé una funcionalidad para copiar el texto cifrado y desencriptarlo con un solo clic. Para mejorar la estética, realicé modificaciones en el diseño propuesto, añadiendo un fondo animado y un footer interactivo que aportan dinamismo y un ambiente digital innovador.

Pagina Web de SuperHéroes Creación de un sitio web dedicado a la temática de superhéroes, con una estructura de tres páginas: una introducción, una dedicada a Iron Man y otra a Spiderman. Utilizando JavaScript, implementé interactividad en las imágenes para mejorar la navegación y la experiencia visual del usuario. El diseño es colorido, con un enfoque en la estética visual mediante la integración de una galería dinámica que exhibe una amplia colección de imágenes de superhéroes, generando una experiencia atractiva y envolvente.
