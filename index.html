Skills


En mi día a día, trabajo con HTML, CSS y JavaScript para desarrollar páginas web funcionales y atractivas. HTML estructura el contenido, CSS mejora la apariencia y JavaScript agrega interactividad. Busco expandir mis habilidades aprendiendo Node para crear aplicaciones del lado del servidor y React para optimizar interfaces y gestionar el estado de manera eficiente. Con estas herramientas, mi objetivo es construir experiencias web más complejas y envolventes, combinando diseño y funcionalidad de forma efectiva.

Mis Proyectos




Encriptador de Texto


Desarrollé un encriptador de texto web con HTML, CSS y JavaScript, que cifra/descifra mensajes, permite copiarlos fácilmente y ofrece una interfaz moderna. Personalicé el diseño con un fondo animado y un footer interactivo, optimizando la experiencia del usuario en un entorno dinámico e innovador.



Historia de los Comics


Diseñé un sitio web de superhéroes con tres páginas (introducción, Iron Man y Spiderman), destacando un diseño colorido y una galería dinámica. Usé JavaScript para añadir interactividad en imágenes, mejorando la navegación y creando una experiencia visual atractiva y envolvente para el usuario.



AluraGeek

Eliminar productos Quitar del listado aquellos productos que ya no desees mostrar



Juego Final del Mundo


Aplicacion creada con orientacion GitHub-Copilot y Javascript. Tomando la Libreria P5.js se desarrolla un juego clasico de Pong con animación, sonido y dibujo.



Fokus


Aplicacion con la tecnica Pomodoro para trabajo y pausa activa. Aplicacion responsiva que usa tecnologias HTML, CSS y JavaScript con cambios de fondo, activación de sonido e inicio de temporizador y click en botones para pausar y reiniciar.



AluraPlay

Con base primaria first Mobile, trabajo basico uso de contenedores, padding, margin, flex-direction, justify-content, align-items, flex-wrap, gap. y demas. Super necesarios para el desarrollo de aplicaciones responsivas.
